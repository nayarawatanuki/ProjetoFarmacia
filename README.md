# Projeto de Farmácia IFSP
Este é um Projeto de Farmácia para a disciplina de Programação Orientada à Objetos. É baseado na arquitetura em três camadas MVC, BCE, etc. e programada em Java.

## Checklist
- [x] CaixaDAO
- [x] CaixaHistoricoDAO
- [x] CartaoDAO
- [x] ClienteDAO
- [x] DinheiroDAO
- [x] FuncionarioDAO
- [x] ItemPedidoDAO
- [ ] LoginDAO
- [x] MedicamentoDAO
- [x] PedidoDAO

## Interação básica

### Atendente
1. Realizar login no sistema
2. Exibir tela de caixa (adicionar item, remover item, conceder desconto, fechar pedido, sair)

### Gerente
1. Login
2. Menu de opções (medicamentos, clientes, funcionário, vendas, pagamentos)
